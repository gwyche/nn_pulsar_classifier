Pulsar Discrimination Success Rate: %


This web application utilizes a fully-connected, multi-layer perceptron neural network to analyze pre-classified radio sources from the High Time Resolution Universe Survey (South). The user can designate the number of layers used to analyze the data. After training, an evaluation test on a subset of datapoints is performed and correct guesses by the trained AI are tabulated. The final percentage of success is displayed above. Training data is stored in a relational database. The full dataset consists of 17800 radio sources; 17400 are used for 3 rounds of training of 52200 and 1300 are used for evaluation.
